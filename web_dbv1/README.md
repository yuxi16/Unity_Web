## Overview

The project uses Unity to connect with MySQL database, it has login and register function in the main page. User can enter correct username and password to login or directly create new user information. The new user information will be stored in MySQL database.<br>

After successfully login, user can input the item name to show the price of the corresponding item. The table is also stored in MySQL database.<br>
